Core Web Vitals

Que mide cada metrica

LCP

Largest Contentful Paint — Carga principal

Tiempo que tarda el elemento mas grande en aparecer. Meta: menos de 2.5s. Impacto directo en el ranking de Google.

CLS

Cumulative Layout Shift — Estabilidad visual

Cuanto se mueve el contenido mientras carga la pagina. Meta: menos de 0.1. Un CLS alto hace que el usuario haga clic en el lugar equivocado.

FCP

First Contentful Paint — Primera carga

Tiempo hasta que aparece el primer elemento visible. Meta: menos de 1.8s. Define la percepcion inicial de velocidad.

TBT

Total Blocking Time — Tiempo bloqueado

Tiempo que el navegador estuvo ocupado sin responder. Meta: menos de 200ms. Afecta la interactividad percibida.
