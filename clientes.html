Core Web Vitals

Qué mide cada métrica

LCP

Largest Contentful Paint

Tiempo que tarda el elemento principal en aparecer. Meta: menos de 2.5s. Impacto directo en el ranking de Google.

CLS

Cumulative Layout Shift

Cuánto se mueve el contenido mientras carga. Meta: menos de 0.1. Afecta la usabilidad y conversión.

FCP

First Contentful Paint

Tiempo hasta el primer elemento visible. Meta: menos de 1.8s. Define la percepción inicial de velocidad.

TBT

Total Blocking Time

Tiempo que el navegador estuvo bloqueado. Meta: menos de 200ms. Afecta la interactividad.
